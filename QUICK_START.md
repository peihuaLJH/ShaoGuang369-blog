## 🚀 启动系统（3 步）

### 1️⃣ 启动后端服务
```bash
npm run server
```
期望输出：
```
Server running on port 5000
MongoDB connected
```

### 2️⃣ 启动前端应用
```bash
npm start
```
系统会自动打开 http://localhost:3000

### 3️⃣ 创建管理员账户（首次）
```bash
node backend/createAdmin.js
```
按提示输入用户名和密码（例如 admin/password123）

## 📖 使用说明

### 登录管理后台
1. 点击网页右上角 **"管理"** 按钮
2. 输入用户名和密码
3. 进入管理面板

### 创建文章
1. 点击 **"创建文章"**
2. 填写标题、摘要、分类、标签
3. 在 Markdown 编辑器中编写内容
4. 点击 **"发布"** 或 **"保存为草稿"**

### 编辑/删除文章
1. 在文章列表中找到你的文章
2. 点击 **"编辑"** 修改或 **"删除"** 移除
3. 更改后点击 **"保存更新"**

### 创建随笔
1. 点击 **"创建随笔"**
2. 填写标题和内容
3. 发布

## 📊 系统架构

```
前端 (React)
    ↓
管理面板 (AdminPanel.js)
    ↓
API 服务 (Express)
    ↓
数据库 (MongoDB)
```

### 关键文件
| 文件 | 作用 |
|------|------|
| `src/components/AdminPanel.js` | 管理界面（已修复） |
| `backend/server.js` | 后端服务 |
| `backend/routes/posts.js` | 文章 API |
| `backend/models/Post.js` | 文章数据模型 |

## 🆘 快速排查

| 症状 | 解决方案 |
|------|--------|
| 登录失败 | 运行 `node backend/createAdmin.js` 创建账户 |
| 无法编辑文章 | 确保后端正在运行 (npm run server) |
| 保存时出错 | 检查浏览器控制台错误，查看后端日志 |
| 页面无信息加载 | 刷新页面，检查 http://localhost:5000/api/posts |
