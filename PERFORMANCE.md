# 性能优化指南

## 1. 前端优化

### 1.1 资源优化

#### 1.1.1 图片优化

- **压缩图片**：使用工具如 TinyPNG、Squoosh 压缩图片
- **响应式图片**：使用 `srcset` 和 `sizes` 属性
- **懒加载**：实现图片懒加载，减少初始加载时间
- **WebP 格式**：使用 WebP 格式图片，减少文件大小

#### 1.1.2 CSS 优化

- **减少 CSS 文件大小**：移除未使用的 CSS
- **CSS 压缩**：使用工具如 cssnano 压缩 CSS
- **关键 CSS**：将关键 CSS 内联到 HTML 中
- **CSS 预加载**：使用 `preload` 预加载重要 CSS

#### 1.1.3 JavaScript 优化

- **代码分割**：使用 React.lazy() 和 Suspense 实现代码分割
- **Tree Shaking**：移除未使用的代码
- **压缩和混淆**：使用 Terser 压缩和混淆 JS
- **延迟加载**：使用 `defer` 或 `async` 属性
- **缓存策略**：合理设置缓存策略

#### 1.1.4 字体优化

- **字体子集**：只包含使用的字符
- **字体预加载**：使用 `preload` 预加载字体
- **系统字体**：优先使用系统字体
- **字体显示策略**：使用 `font-display` 属性

### 1.2 渲染优化

#### 1.2.1 减少重排和重绘

- **避免频繁 DOM 操作**：批量操作 DOM
- **使用 CSS transform**：代替位置属性
- **使用 will-change**：提示浏览器优化
- **避免布局抖动**：固定元素尺寸

#### 1.2.2 优化首屏加载

- **骨架屏**：显示骨架屏减少感知加载时间
- **渐进式加载**：先加载核心内容
- **服务器端渲染**：考虑使用 Next.js 等 SSR 框架

### 1.3 网络优化

#### 1.3.1 HTTP/2 和 HTTP/3

- **启用 HTTP/2**：减少连接开销
- **考虑 HTTP/3**：进一步提高性能

#### 1.3.2 CDN 加速

- **使用 CDN**：分发静态资源
- **选择合适的 CDN 节点**：靠近用户的节点
- **缓存策略**：合理设置 CDN 缓存

#### 1.3.3 资源缓存

- **HTTP 缓存**：设置合理的缓存头
- **Service Worker**：实现离线缓存
- **本地存储**：缓存频繁访问的数据

## 3. Nginx 优化

### 3.1 配置优化

#### 3.1.1 基本配置

```nginx
# 开启 gzip 压缩
gzip on;
gzip_comp_level 6;
gzip_types text/plain text/css application/json application/javascript text/xml application/xml application/xml+rss text/javascript;

# 缓存配置
location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
    expires 30d;
    add_header Cache-Control "public, max-age=2592000";
}

# 限制请求大小
client_max_body_size 10M;

# 超时设置
keepalive_timeout 65;
```

#### 3.1.2 高级配置

- **启用 HTTP/2**
- **配置 SSL 会话缓存**
- **设置适当的 worker 进程数**
- **启用 TCP 快速打开**

### 3.2 安全优化

- **启用 HTTPS**
- **配置 HSTS**
- **设置 CSP 头**
- **防止点击劫持**

## 9. 常见性能问题及解决方案

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 页面加载缓慢 | 资源过大、过多 | 压缩资源、代码分割、CDN |
| API 响应慢 | 数据库查询慢、逻辑复杂 | 优化查询、缓存、异步处理 |
| 数据库性能差 | 缺少索引、查询不当 | 创建索引、优化查询、分页 |
| 服务器负载高 | 流量过大、资源不足 | 负载均衡、水平扩展、优化代码 |
| 内存泄漏 | 代码问题、资源未释放 | 代码审查、内存监控、修复泄漏 |
| 带宽占用高 | 大文件传输、流量过大 | 压缩、CDN、缓存 |
